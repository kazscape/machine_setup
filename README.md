windows_setting
====

Set up windows

## Description
Set up windows as follows
- Install scoop
- Install application by scoop
  - visual studio code
  - Google Chrome
  - Slack
  - Powershell core
  - Windows Terminal
  - Winmerge
  - Zoom
- Install fonts
- Modify "settings.json" for Windows Terminal
- Install oh-my-posh
- Set up WSL
- Install Ubuntu distribution

Also, Can uninstall above settings
